# BasisPantry 52 – Verwendung

Dieses Verzeichnis enthält ein vorkonfiguriertes Pantry-Preset für nutritionOS.

## Grundprinzip

nutritionOS erlaubt es ausdrücklich, eigene Lebensmittel manuell zu erfassen und zu hinterlegen.  
Wer das tut, **benötigt dieses Preset nicht**.

Dieses Preset richtet sich an Nutzer:innen, die ihre Vorräte **nicht selbst erfassen möchten**  
und stattdessen mit einer fertigen, alltagstauglichen Basis arbeiten wollen.

## Verwendung des Presets

Wenn dieses Pantry-Preset geladen ist, geht nutritionOS davon aus,  
dass die hier hinterlegten Lebensmittel **grundsätzlich verfügbar** sind.

Auf dieser Annahme basierend:
- entwirft das System Rezepte
- schlägt Mahlzeiten vor
- kombiniert Zutaten zu alltagstauglichen Gerichten

Damit das sinnvoll funktioniert, sollten die unten aufgeführten Lebensmittel
tatsächlich im Vorratsschrank vorhanden sein.

Die Liste kann als **Einkaufszettel** verwendet werden.

## Anpassung des Presets

Dieses Preset kann angepasst werden.

Wenn ein Lebensmittel nicht verwendet werden soll:

1. Den entsprechenden Eintrag aus dem **Inhaltsverzeichnis** entfernen  
2. Den zugehörigen Lebensmitteleintrag mit Nährwerten  
   in der Lebensmitteldatei vollständig löschen

Nach der Entfernung geht nutritionOS davon aus,
dass dieses Lebensmittel **nicht verfügbar** ist
und berücksichtigt es entsprechend nicht mehr.
